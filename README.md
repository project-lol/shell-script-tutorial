## What I Learned

### touch

- 파일을 생성하는 명령어

```bash
touch [파일명]
```

<br>

### chmod

- 파일의 권한을 변경하는 명령어
- 이 명령어를 통해서 파일을 실행할 수 있게 만들 수 있다

```bash
chmod +x myscript.sh
```

- `+x`는 실행 권한을 부여한다는 의미이다
- `+x` 대신 `-x`를 사용하면 실행 권한을 제거할 수 있다
